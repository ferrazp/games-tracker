## 🎯 Visión General

Este es el **backend** del proyecto Games Tracker, un gestor personal de videojuegos jugados. El proyecto está diseñado para ser **dual-database** (SQLite + PostgreSQL), permitiendo:

- **SQLite**: Para consumo personal y desarrollo local
- **PostgreSQL**: Para producción/despliegue público

El frontend se encuentra en: `F:\projects\developments\games-tracker` ([AGENTS.md del frontend](F:\projects\developments\games-tracker\AGENTS.md))
