## 🏗️ Stack Tecnológico

| Component | Tecnología | Propósito |
|-----------|-----------|----------|
| **Runtime** | Node.js 20+ | Servidor backend |
| **Framework** | Express.js | API REST |
| **BD (Local)** | SQLite3 | Desarrollo personal |
| **BD (Prod/Dev Docker)** | PostgreSQL 17 | Producción y desarrollo Docker |
| **Package Managers** | npm + pnpm | Gestión de dependencias |
| **Puerto API (local)** | 4000 | Desarrollo sin Docker |
| **Puerto API (dev Docker)** | 4001 | Docker dev |
| **Puerto API (prod Docker)** | 4001 | Docker prod |
| **Puerto Front (local)** | 3000 | Frontend standalone |
| **Puerto Front (dev Docker)** | 3001 | Docker dev |
| **Puerto Front (prod Docker)** | 9090 | Docker prod |
| **External API** | Twitch IGDB | Búsqueda online (opcional) |
| **Orquestación** | Docker Compose | Contenedores |
| **Proxy** | Nginx 1.27 | Frontend + reverse proxy a API |
