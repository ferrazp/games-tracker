## 🏗️ Stack Tecnológico

| Component | Tecnología | Propósito |
|-----------|-----------|----------|
| **Runtime** | Node.js v18+ | Servidor backend |
| **Framework** | Express.js | API REST |
| **BD (Local)** | SQLite3 | Desarrollo personal |
| **BD (Prod)** | PostgreSQL 16 | Producción/público |
| **Package Managers** | npm + pnpm | Gestión de dependencias |
| **Port** | 4000 | API (configurable en `.env`) |
| **External API** | Twitch IGDB | Búsqueda online (opcional, fallback) |
